# pdffill

CLI for listing, diagnosing, diffing, and filling PDF AcroForms.

```bash
npm install -g pdffill
pdffill doctor template.pdf
pdffill run template.pdf data.json -o filled.pdf
```

Requires Node.js 18+. Library: [`@pdffill/core`](https://www.npmjs.com/package/@pdffill/core).

Documentation: [README](https://github.com/pdffill/pdffill/blob/main/README.md) (update repo URL if yours differs).
